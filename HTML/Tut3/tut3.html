Types of elements in HTML: 1. Block level elements 2. Inline elements Block level elements: 1. Block level elements always start on a new line. 2. Block level elements take up the full width available. 3. Block level elements can have margins and padding. 4. Block level elements can have height and width set. 5. Block level elements can have a background color set. 6. Block level elements can have borders set. 7. Block level elements can be nested inside other block level elements. 8. Block level elements can wrap inline elements. 9. Block level elements can wrap other block level elements. Inline elements: 1. Inline elements do not start on a new line. 2. Inline elements only take up as much width as necessary. 3. Inline elements do not have margins or padding. 4. Inline elements do not have height and width set. 5. Inline elements do not have a background color set. 6. Inline elements do not have borders set. 7. Inline elements can be nested inside other inline elements. 8. Inline elements can wrap inline elements. 9. Inline elements cannot wrap block level elements.
